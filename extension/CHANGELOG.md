# Extension Changelog

## Version 1.0.0 (Initial Release)

### Features
- ✅ Quick save any webpage to your Stack collections
- ✅ Beautiful dark theme matching Stack website
- ✅ Rich previews with metadata (title, description, image)
- ✅ Instant favicon display
- ✅ Keyboard shortcuts (Enter to save, Esc to close)
- ✅ Error handling for protected pages
- ✅ Optimized metadata fetching (cache-first, then queue)
- ✅ Smooth animations and transitions
- ✅ Accessible UI with proper ARIA labels

### Technical
- Manifest V3 compliant
- CORS-enabled API endpoints
- Rate limiting protection
- Secure authentication flow
- Cross-browser compatible (Chrome, Edge)

### UI/UX Improvements
- Loading states with spinner
- Error messages with clear feedback
- Preview card with image support
- Responsive design
- Smooth fade-in animations
- Hover effects on interactive elements
- Focus states for keyboard navigation
