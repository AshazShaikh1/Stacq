# Icon Generation Instructions

The extension requires three PNG icon files:
- `icon-16.png` (16×16 pixels)
- `icon-48.png` (48×48 pixels)  
- `icon-128.png` (128×128 pixels)

## Option 1: Use the HTML Generator (Easiest) ⭐

1. Open `generate-icons.html` in your browser
2. Click each "Download" button
3. Save the files to the `icons/` folder
4. Done!

## Option 2: Use Online Tools

1. Go to https://convertio.co/svg-png/ or similar
2. Upload `icons/icon.svg`
3. Convert to PNG
4. Resize to 16×16, 48×48, 128×128
5. Save as `icon-16.png`, `icon-48.png`, `icon-128.png`

## Option 3: Create Simple Placeholders

If you just want to test the extension quickly:

1. Create a simple image editor (Paint, GIMP, etc.)
2. Create a 16×16 square with cyan background (#14E1D9)
3. Add a white or dark "S" letter
4. Save as PNG
5. Resize to create all three sizes

## Icon Design

- **Background**: Electric Cyan (#14E1D9)
- **Letter**: Dark (#071017) or White
- **Shape**: Rounded rectangle (18px radius for 128px icon)
- **Font**: Bold, centered

The icon should be recognizable at small sizes, so keep it simple!
